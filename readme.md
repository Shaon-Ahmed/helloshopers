## Installation Instruction

- Clone the .repo
- Run 'composer install'
- 
- Run 'php artisan migrate --seed'

## Contribution Guideline

- Fork the repo
- Clone repo locally
- Run 'git checkout dev'
- Create a new branch
- Work on your local branch
- push to  remote
- When work is tested or done, push to remote
- Merge to dev
